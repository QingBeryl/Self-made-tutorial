## 注释

1. 单行注释

   ```java
   //打印HelloWorld
   public class HelloWorld{
   	public static void main(String[] args){
   		System.out.println("你好");
   	}
   }
   ```

   ```java

   }
   ```

## 字面量

| 字面量类型 | 说明                                   | 举例        |
| ---------- | -------------------------------------- | ----------- |
| 整数       | 不带小数点的数字                       | 666         |
| 小数       | 带小数点的数字                         | 133.344     |

## 变量

### 变量定义格式：

数据类型  变量名  =  变量值

```java
int a = 3；
```

### 变量使用方法：

1. 输出打印

   ```java
   int a = 3;
   System.out.println(a);
   ```

2. 参与计算

   ```java
   int a = 20；
   int b = 10;
   System.out.println(a+b);
   ```

3. 修改变量值

   ```java
   int a = 20；
   System.out.println(a);
   a = 10;
   System.out.println(a);
   ```

### 变量的注意事项

- 只能存一个值
- 变量名不允许重复定义
- 一条语句可以定义多个变量
- 变量在使用之前一定要进行赋值
- 变量的作用域范围

## 数据类型

| 数据类型 | 关键字  | 取值范围                                          | 内存占用 |
| -------- | ------- | ------------------------------------------------- | -------- |
| 整数     | byte    | -128~127                                          | 1        |
| 整数     | short   | -32768~32767                                      | 2        |
| 整数     | int     | -2147483648~2147483647(10位数)                    | 4        |
| 整数     | long    | -9223372036854775808~9223372036854775807 (19位数) | 8        |
| 浮点数   | float   | -3.401298e-38~3.402823e+38                        | 4        |
| 浮点数   | double  | -4.9000000e-324~1.797693e+308                     | 8        |
| 字符     | char    | 0~65535                                           | 2        |
| 布尔     | boolean | true，flase                                       | 1        |

如果要定义long类型的变量，在数据值的后面要加上一个L（大写，与数字1区分开）作为后缀：

```java
long n = 9999999999L;
System.out.println(n);
```

如果要定义float类型的变量，在数据值的后面要加上一个F（大小写均可）作为后缀：

```java
long f = 9999999999F;
System.out.println(f);
```

整数和小数取值范围大小关系:
double> float> long>int> short> byte

## 标识符

标识符：就是给**类、方法、变量**等起的名字。

1. 硬性要求：
   - 由数字、字母、下划线(_)和美元符($)组成
   - 不能以数字开头
   - 不能是关键字
   - 区分大小写
2. 软性要求：
   - 小驼峰命名法：方法、变量
     - 规范1:标识符是一个单词的时候，全部小写
     - 规范2:标识符由多个单词组成的时候，第一个单词首字母小写，其他单词首字母大写
   - 大驼峰命名法：类
     - 规范1:标识符是一个单词的时候，首字母大写
     - 规范2:标识符由多个单词组成的时候，每个单词
3. 无论如何，须满足见名知意的要求。

## 键盘录入

Java帮我们写好一个类叫Scanner，这个类就可以接收键盘输入的数字。

- 步骤一:导包---Scanner这个类在哪

  ```java
  import java.util.Scanner;
  ```

  导包的动作必须出现在类定义的上边。

- 步骤二:创建对象---表示我要开始用Scanner这个类了

  ```java
  Scanner sc = new Scanner(System.in);
  ```

  上面这个格式里面，只有sc是变量名，可以变，其他的都不允许变。

- 步骤三:接收数据---真正开始干活了

  ```java
  int i=sc.nextInt();
  ```

  左面这个格式里面，只有i是变量名，可以变，其他的都不允许变。
